


# Advanced Model Selector

A small UI showcase: an AI model picker built into a chat-style input.

Search models, preview details on hover, tweak settings (reasoning, speed), and navigate with the keyboard. Model names and metrics are fake — this is front-end UX only.

## Run locally

```bash
npm install
npm run dev
```

## Stack

React, TypeScript, Vite, Tailwind, Base UI, Floating UI, Motion.
